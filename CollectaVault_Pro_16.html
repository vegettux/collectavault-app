Conectar GitHub (3 minutos)

  1. 1
    Ve a github.com → Crea una cuenta gratuita si no tienes → Crea un repositorio privado llamado collectavault-data
  2. 2
    Tu avatar → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → activa solo el permiso repo → copia el token (ghp_...)
  3. 3
    Rellena y pulsa Conectar
